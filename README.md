# 🌍Global Air Quality Monitor

A web application that fetches and displays real-time air quality 
data for any city using the OpenWeatherMap API.

## Features
- Search any city to get live AQI data
- Displays pollution levels in a clean table format
- Compares air quality across multiple cities
- Tracks previously searched cities using search history

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- API: OpenWeatherMap Air Pollution API

## How It Works
1. User enters a city name in the search box
2. App fetches real-time AQI data from OpenWeatherMap API
3. Pollution levels are displayed in a structured table
4. Search is saved to history for future reference
